# Окружение и инфраструктура

Краткая справка, чтобы не уточнять каждый раз.

## Home Assistant

- **Установка:** HAOS (Home Assistant Operating System).
- **Где крутится:** виртуальная машина в **Proxmox**.
- **Сеть:** в одной локальной сети с остальными устройствами (телефоны, приставки и т.д.).

## Устройства (по мере появления в задачах)

- **Ugoos X4Q** — Android-приставка (обычный Android, arm64). Используется в т.ч. для трансляции экрана в HA.
