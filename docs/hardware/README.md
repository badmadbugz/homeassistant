# Железо (hardware)

Устройства, датчики, шлюзы — что где стоит, как подключено, заметки.

---

## Как организован раздел

Сейчас один индексный файл. Дальше можно развивать в одном из форматов:

| Подход | Папки/файлы | Плюсы | Минусы |
|--------|-------------|--------|--------|
| **По комнатам** | `комнаты/КОРИДОР.md`, `комнаты/КУХНЯ.md` | «Что в этой комнате» — быстро найти по месту | Один девайс в нескольких комнатах (шлюз) — дублирование |
| **По типу** | `шлюзы.md`, `датчики.md`, `свет.md`, `кнопки_пульты.md`, `прочее.md` | Удобно смотреть «все датчики» или «все лампы» | Комната неочевидна |
| **По интеграции** | `zigbee.md`, `esphome.md`, `wi-fi.md` | Удобно при отладке «что с Zigbee» | Один файл может разрастаться |
| **Сводная таблица + файлы по необходимости** | `README.md` (таблица) + отдельный файл только для сложных/важных устройств (Ugoos, шлюз, Fingerbot) | Мало файлов, детали только там, где нужно | Таблица нужно поддерживать в актуальном состоянии |

**Рекомендация:** начать со **сводной таблицы в этом README** и добавлять отдельные файлы (например `Ugoos.md`, `Zigbee_шлюз.md`, `Fingerbot.md`) только для устройств, по которым есть что подробно расписать (подключение, прошивка, батарейки, сценарии).
